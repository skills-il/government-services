# Bituach Leumi Benefit Programs Reference

## All Programs

| Program | Hebrew | Eligibility Summary |
|---------|--------|-------------------|
| Old Age Pension | kiztavat zikna | Men 67, Women 62-65, qualifying contributions |
| Disability | nechut | Medical disability > 40% |
| Unemployment | dmei avtala | 12+ months employed, terminated |
| Maternity | dmei leida | 6-15 months employed |
| Child Allowance | kiztavat yeladim | All residents with children under 18 |
| Work Injury | pgiat avoda | Workplace injury/illness |
| Reserve Duty | miluim | IDF reserve service |
| Income Support | havtachat hachnasa | Below minimum income |
| Long-term Care | siyud | Elderly needing daily assistance |
| Survivor Pension | sheerim | Widows/widowers, orphans |
| Mobility Allowance | niyadut | Mobility-impaired individuals |
| Alimony | mezonot | Court-ordered, unpaid |
| Hostile Action Victims | nifgaei peulot eiiva | Terror/war victims |
