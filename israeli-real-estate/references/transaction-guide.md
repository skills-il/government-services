# Israeli Real Estate Transaction Guide

## Buying Process Checklist

1. Pre-approval (ishur ikroni) from bank
2. Property search and viewings
3. Hire real estate attorney BEFORE signing
4. Attorney obtains Tabu extract
5. Price negotiation and payment schedule
6. Sign purchase agreement (chozeh mcher)
7. Pay purchase tax (mas rechisha) within 50 days
8. Finalize mortgage (mashkanta) with bank
9. Attorney registers transfer in Tabu
10. Key handover per contract

## Purchase Tax Brackets (2025)

### First Apartment
| Range (NIS) | Rate |
|-------------|------|
| 0 - 1,978,745 | 0% |
| 1,978,746 - 2,347,040 | 3.5% |
| 2,347,041 - 6,055,070 | 5% |
| 6,055,071 - 20,183,565 | 8% |
| 20,183,566+ | 10% |

### Non-First Apartment
| Range (NIS) | Rate |
|-------------|------|
| 0 - 6,055,070 | 8% |
| 6,055,071 - 20,183,565 | 10% |
| 20,183,566+ | 12% |

## Tabu Extract Sections
1. Property description (gush, chelka)
2. Registered owners and shares
3. Mortgages (mashkantaot)
4. Liens, warnings, court orders

## Key Costs
- Purchase tax: See brackets above
- Attorney: 0.5-1.5% of price
- Agent: 1-2% + VAT
- Mortgage fees: Application, appraisal, registration
- Arnona transfer: Municipal property tax
