# סטטיסטיקה ישראלית (למ"ס)

## שימו לב
נתונים סטטיסטיים מתפרסמים בלוח זמנים קבוע עם השהייה מובנית. מדד המחירים
לצרכן מתפרסם ב-15 בחודש העוקב בשעה 18:30. מדד מחירי דירות מתפרסם חודשית עם
פיגור של כשישה שבועות (ויש גם דו"ח עסקאות רבעוני מפורט יותר). תמיד תציינו את
תקופת הייחוס כשאתם מציגים נתונים. ערכי מדדים ושיעורים משתנים, תאמתו נתונים
עדכניים באתר הלמ"ס (cbs.gov.il) להחלטות תלויות-זמן כמו עדכוני שכר דירה או
הצמדה חוזית.

## תמונת מצב (נכון למאי 2026)

את ערך המדד העדכני ב-`api.cbs.gov.il/index/data/price?id=120010` לפני החישוב.

לשליפה חוזרת לפני קבלת החלטה תלוית ערך עדכני.

### שלב 2: מדד המחירים לצרכן -- "המדד"
מדד המחירים לצרכן הוא המדד המצוטט ביותר בישראל.

**איך להשתמש בנתוני המדד:**
1. **מדד חודשי נוכחי:** הערך האחרון שפורסם והשינוי החודשי
2. **שינוי שנתי:** שינוי באחוזים שנה-על-שנה (שיעור האינפלציה)
3. **פירוט לפי רכיבים:** אילו מגזרים מניעים את שינויי המחירים

**רכיבי מדד המחירים לצרכן (משקלות משוערים):**
| רכיב | משקל (~%) |
|------|-----------|
| דיור (שכר דירה) | ~25% |
| תחבורה | ~17% |
| מזון | ~16% |
| בריאות | ~6% |
| חינוך ותרבות | ~8% |
| הלבשה והנעלה | ~3% |
| ריהוט ומשק בית | ~5% |
| אחר | ~20% |

**נוסחת עדכון שכר דירה (לחוזים צמודי מדד):**
```
שכ"ד חדש = שכ"ד ישן * (מדד נוכחי / מדד בעת חתימת החוזה)
```
דוגמה: אם המדד עלה מ-100.0 ל-103.5 מאז תחילת החוזה, שכר הדירה עולה ב-3.5%.

### שלב 3: מדד מחירי דירות
עוקב אחר מחירי עסקאות דירות:

**חתכים עיקריים:**
- **ממוצע ארצי:** מגמת מחירים כללית לכל ישראל
- **לפי מחוז:** ירושלים, תל אביב, חיפה, מרכז, דרום, צפון
- **לפי עיר:** ערים מרכזיות נעקבות בנפרד
- **לפי סוג נכס:** דירות חדשות לעומת דירות יד שנייה
- **לפי גודל דירה:** 1.5-2, 2.5-3, 3.5-4, 4.5+ חדרים

**פירוש המדד:**
- שינוי רבעוני: כיוון שוק לטווח קצר
- שינוי שנתי: מגמה לטווח בינוני (מחליק השפעות עונתיות)
- בהשוואה לשכר: מדד ליכולת רכישה

### שלב 4: לוח מדדים כלכליים
נתונים מקרו-כלכליים מרכזיים מהלמ"ס:

**תמ"ג (תוצר מקומי גולמי):**
- פרסום רבעוני, מנוכה עונתיות
- שיעורי צמיחה ריאליים לעומת נומינליים
- תמ"ג לנפש להשוואה בינלאומית

**שוק העבודה:**
- שיעור אבטלה: חודשי, גילאי 15+
- שיעור השתתפות בכוח העבודה
- תעסוקה לפי ענף (הייטק, תעשייה, שירותים וכו')
- שכר ממוצע לפי ענף (מתפרסם רבעונית)

**סחר חוץ ומאזן תשלומים:**
- נתוני יצוא/יבוא חודשיים
- פירוט סחורות לעומת שירותים
- שותפי סחר עיקריים

**בנייה:**
- התחלות בנייה: מדד מוביל להיצע דירות
- גמירות בנייה
- צינור היתרים שהונפקו לעומת התחלות לעומת גמירות

### שלב 5: נתונים דמוגרפיים
הלמ"ס הוא המקור המוסמך לנתונים דמוגרפיים בישראל:

**אוכלוסייה:**
- אוכלוסייה כוללת: מתעדכנת רבעונית (ביניים) ושנתית (סופי)
- לפי דת: יהודים, מוסלמים, נוצרים, דרוזים, אחר
- לפי מחוז ועיר
- התפלגות גילאים ויחסי תלות
- תחזיות אוכלוסייה (טווח בינוני וארוך)

**הגירה:**
- סטטיסטיקת עלייה: חודשית לפי ארץ מוצא
- אומדני ירידה
- מגמות הגירה נטו

**סטטיסטיקה חיונית:**
- שיעור לידות, שיעור תמותה, ריבוי טבעי
- תוחלת חיים (לפי מגדר)
- שיעורי נישואין וגירושין

### שלב 6: שאילתת נתוני הלמ"ס
באמצעות שרת ה-MCP israel-statistics או גישה ישירה ללמ"ס.

**שני מקורות נתונים נפרדים:**
- ממשק מדדי המחירים של הלמ"ס (`api.cbs.gov.il/index`) הוא המקור הקנוני למדד המחירים לצרכן, מחירי דירות, מדד מחירי יצרן ומדד תשומות הבנייה. רשימת המדדים: `api.cbs.gov.il/index/catalog/catalog?format=json`, ושליפת סדרה: `api.cbs.gov.il/index/data/price?id=<קוד>&format=json` (קוד מדד המחירים הוא `120010`, מחירי דירות `40010`).
- אתר data.gov.il תחת הארגון `lamas` (ולא `cbs`) מאחסן מספר מצומצם של מאגרי נתונים של הלמ"ס, כמו לוחות מפקד, יישובים ותאונות דרכים. הוא אינו מאחסן את סדרות הזמן של מדד המחירים, התמ"ג או האבטלה.
- סדרות תמ"ג, אבטלה, אוכלוסייה וסחר חוץ מתפרסמות כטבלאות ממוספרות של הלמ"ס באתר `cbs.gov.il` ולא כולן זמינות דרך API ציבורי.

**מבנה טבלאות הלמ"ס:**
- טבלאות מזוהות לפי מספר (למשל, טבלה 2.1, טבלה 12.5)
- נתונים מאורגנים לפי תחום (אוכלוסייה, מחירים, עבודה וכו')
- זמינים בעברית ובאנגלית חלקית
- ניתנים להורדה כ-CSV, Excel, או דרך API

**מפתח טבלאות נפוצות:**
| נושא | טווח טבלאות | תיאור |
|------|------------|-------|
| אוכלוסייה | 2.x | גודל אוכלוסייה, דמוגרפיה |
| הגירה | 4.x | עלייה וירידה |
| מחירים | 12.x | מדד המחירים, דיור, מחירי יצרן |
| עבודה | 12.x | תעסוקה, שכר, אבטלה |
| חשבונות לאומיים | 16.x | תמ"ג, צמיחה, לנפש |
| בנייה | 19.x | פעילות בנייה |
| סחר חוץ | 16.x | יצוא, יבוא |

**טיפים לשאילתות נתוני הלמ"ס:**
- שמות שדות בעברית נפוצים -- יש לבדוק כותרות עמודות
- פורמטי תאריך משתנים: חלק מהטבלאות משתמשות בלוח העברי, רובן בלוח הגרגוריאני
- ניכוי עונתיות: סדרות רבות זמינות הן גולמיות והן מנוכות עונתיות
- עדכונים: נתונים ראשוניים עשויים להתעדכן בפרסומים עוקבים

## דוגמאות

### דוגמה 1: מדד ועדכון שכר דירה
המשתמש אומר: "בעל הדירה רוצה להעלות לי את השכירות לפי המדד. זה מותר?"
תוצאה: הסבר חוזי שכירות צמודי מדד. אם החוזה מציין עדכון לפי מדד המחירים, יש לחשב: חיפוש המדד בתאריך חתימת החוזה והמדד הנוכחי, יישום הנוסחה. ציון שעדכונים הם בדרך כלל שנתיים ולא חודשיים. אם המדד ירד, שכר הדירה צריך לרדת גם כן.

### דוגמה 2: ניתוח שוק הדיור
המשתמש אומר: "מחירי דירות בתל אביב עולים או יורדים?"
תוצאה: שאילתת מדד מחירי דירות למחוז תל אביב. הצגת מגמות רבעוניות ושנתיות. השוואה לממוצע הארצי. ציון הקשר רלוונטי: ריביות, התחלות בנייה באזור, גורמי היצע/ביקוש.

### דוגמה 3: סקירה כלכלית
המשתמש אומר: "מה המצב של הכלכלה הישראלית?"
תוצאה: הצגת צמיחת תמ"ג אחרונה (רבעונית, מתואמת שנתית), שיעור אבטלה, שיעור אינפלציה במדד המחירים, מגמות שער השקל, וביצועי ענפים בולטים. ציון מקורות הלמ"ס לכל נתון. נכון למאי 2026 נקודת היחס: תמ"ג צמח 3.0% ב-2025, אינפלציה שנתית 1.9% באפריל 2026, אבטלה 2.7% בפברואר 2026, ריבית בנק ישראל 4.00% (נקבעה 30.3.2026). תמיד יש לאמת ערכים מול הלמ"ס בעת המענה.

## משאבים מצורפים

### סקריפטים
- `scripts/fetch_cbs_data.py` - שאילתות מול ממשק מדדי המחירים של הלמ"ס (`api.cbs.gov.il`): שליפת ערכי מדד המחירים (המדד) האחרונים יחד עם משקלות הרכיבים, חיפוש בקטלוג המדדים, חישוב עדכוני שכר דירה צמודי מדד לפי ערכי מדד ישנים/חדשים, והצגת סיכום מדדים כלכליים מרכזיים עם המקור הנכון לכל סדרה. תומך בפקודות: `cpi`, `rent-calc`, `search`, `indicators`. הרצה: `python scripts/fetch_cbs_data.py --help`

### חומרי עזר
- `references/cbs-data-guide.md` - לוח פרסומי הלמ"ס לכל המדדים המרכזיים (מדד המחירים, מחירי דירות, תמ"ג, אבטלה, התחלות בנייה), משקלות רכיבי מדד המחירים, נוסחת עדכון שכר דירה לחוזים צמודי מדד, ומפתח מספרי טבלאות הלמ"ס לפי נושא (אוכלוסייה 2.x, מחירים 12.x, בנייה 19.x וכו'). יש לעיין במסמך בעת קביעת זמינות נתונים או איתור מספר טבלת הלמ"ס הנכון.

## מלכודות נפוצות
- הלשכה המרכזית לסטטיסטיקה (למ"ס) מפרסמת נתונים בעברית בעיקר. תגובות API ומטא-נתונים משתמשים בשמות שדות בעברית. סוכנים עלולים להיכשל בפענוח שמות עמודות לא-ASCII.
- סקרים סטטיסטיים ישראליים משתמשים במערכת סיווג גיאוגרפי שונה (נפה, מחוז) ממדינות/מחוזות בארה"ב. סוכנים עלולים לנסות למפות אזורים ישראליים למושגים גיאוגרפיים אמריקאיים.
- לוחות זמני פרסום של הלמ"ס קבועים אבל הפרסומים לעתים כוללים נתונים ראשוניים שמתוקנים בפרסומים עוקבים. סוכנים עלולים להציג נתונים ראשוניים כסופיים בלי לציין את סטטוס העדכון.
- נתוני אוכלוסייה בישראל כוללים או מחריגים שטחים שונים בהתאם למערך הנתונים. סוכנים צריכים לאמת האם נתון מסוים מכסה את ישראל, יישובי יהודה ושומרון, או מזרח ירושלים.

פתרון: יש לבדוק את לוח הפרסומים של הלמ"ס לתאריך הפרסום הצפוי. מדד המחירים: ~ה-15 לחודש. מחירי דירות: ~6 שבועות לאחר סיום רבעון. תמ"ג: ~6 שבועות לאחר סיום רבעון.

### שגיאה: "Index base period mismatch"
סיבה: הלמ"ס מעדכנת תקופתית את בסיס המדדים, מה שגורם לשברי סדרות
פתרון: יש לוודא ששני הערכים המושווים משתמשים באותה תקופת בסיס. הלמ"ס מפרסמת טבלאות המרה בין תקופות בסיס ישנות וחדשות. למדד המחירים, יש לבדוק איזו שנת בסיס חלה על הסדרה.

### שגיאה: "Hebrew column names in data"
סיבה: טבלאות נתוני הלמ"ס משתמשות בעיקר בכותרות בעברית
פתרון: שימוש בשרת ה-MCP israel-statistics שיכול לפרש שמות שדות בעברית. לחלופין, שאילתה עם limit=1 לבחינת מבנה העמודות לפני שליפת נתונים מלאה.
