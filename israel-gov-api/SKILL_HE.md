# ממשקי API ממשלתיים - ישראל

## הוראות

### שלב 1: הבנת הצורך במידע
שאלו את המשתמש:
- **באיזה נושא?** (תחבורה, בריאות, חינוך, כלכלה וכו')
- **באיזה אזור גיאוגרפי?** (ארצי, עיר/אזור ספציפי, כתובת מסוימת)
- **באיזו תקופה?** (נוכחי, היסטורי, סדרה עתית)
- **באיזה פורמט?** (נתונים גולמיים, סטטיסטיקה מסכמת, הדמיה חזותית)

### שלב 2: חיפוש מערכי נתונים
השתמשו ב-API של data.gov.il (מבוסס CKAN) לאיתור מערכי נתונים רלוונטיים:

**חיפוש לפי מילת מפתח:**
```
GET https://data.gov.il/api/3/action/package_search?q=KEYWORD&rows=10
```

**חיפוש לפי ארגון (משרד ממשלתי):**
```
GET https://data.gov.il/api/3/action/package_search?fq=organization:MINISTRY_ID
```

**מזהי ארגונים נפוצים:**
| משרד | מזהה | שם באנגלית |
|------|------|------------|
| הלשכה המרכזית לסטטיסטיקה | cbs | Central Bureau of Statistics |
| משרד התחבורה | mot | Ministry of Transportation |
| משרד הבריאות | moh | Ministry of Health |
| משרד החינוך | moe | Ministry of Education |
| רשות המסים | ita | Israel Tax Authority |
| רשות מקרקעי ישראל | ila | Israel Land Authority |
| משרד הפנים | moi | Ministry of Interior |

### שלב 3: שליפה ושאילתת נתונים
לאחר איתור מערך הנתונים:

**קבלת פרטי מערך נתונים:**
```
GET https://data.gov.il/api/3/action/package_show?id=DATASET_ID
```

**שאילתת נתונים טבלאיים (datastore):**
```
GET https://data.gov.il/api/3/action/datastore_search?resource_id=RESOURCE_ID&limit=100
```

**שאילתות דמויות SQL (חזקות):**
```
GET https://data.gov.il/api/3/action/datastore_search_sql?sql=SELECT * FROM "RESOURCE_ID" WHERE field = 'value' LIMIT 100
```

**טיפים:**
- שמות השדות לרוב בעברית — השתמשו ב-`datastore_search` עם `limit=1` תחילה כדי לראות את שמות השדות
- מערכי נתונים גדולים: השתמשו ב-`limit` ו-`offset` לדפדוף
- שדות תאריך עשויים להיות בפורמטים שונים — בדקו את תיעוד מערך הנתונים

### שלב 4: ניתוח והצגה
עבור הנתונים שנשלפו:
1. סכמו את הממצאים העיקריים בשפה פשוטה
2. חשבו סטטיסטיקות בסיסיות אם התבקשו (ממוצע, חציון, מגמות)
3. הציעו הדמיות חזותיות (תרשים עמודות, גרף קווי, מפה) המתאימות לנתונים
4. ציינו את עדכניות הנתונים (תאריך עדכון אחרון) וכל הסתייגות רלוונטית
5. ספקו קישור ישיר למערך הנתונים באתר data.gov.il לעיון

### שלב 5: הצלבת נתונים (מתקדם)
בעת שילוב מערכי נתונים מרובים:
1. זהו מפתחות משותפים (קוד יישוב, תאריך, קוד קטגוריה)
2. השתמשו בקודים מנהליים ישראליים (קודי יישוב של הלמ"ס) לחיבור גיאוגרפי
3. שימו לב ששמות שדות בין מערכי נתונים שונים עשויים להיות שונים — התאימו לפי תוכן ולא לפי שם
4. תעדו את מקור הנתונים: אילו מערכי נתונים תרמו לניתוח

## דוגמאות

### דוגמה 1: חיפוש נתוני בתי ספר
המשתמש אומר: "אני צריך נתונים על בתי ספר בתל אביב"
פעולות:
1. חיפוש: `package_search?q=schools+tel+aviv`
2. איתור מערך נתוני חינוך וקבלת מזהה המשאב
3. שאילתה: סינון לפי קוד יישוב של תל אביב (5000)
4. הצגה: מספר בתי ספר, סוגים, גדלים
תוצאה: נתוני בתי ספר מובנים עבור תל אביב

### דוגמה 2: ניתוח מחירי דיור
המשתמש אומר: "הראה לי מגמות מחירי דיור בחיפה"
פעולות:
1. איתור מערך נתוני עסקאות נדל"ן של רשות המסים
2. סינון לפי קוד יישוב חיפה, 12 חודשים אחרונים
3. חישוב מחיר חציוני למטר רבוע לפי חודש
4. הצגת מגמה עם אחוזי שינוי
תוצאה: מגמת מחירים חודשית לחיפה עם ניתוח

## משאבים מצורפים

### סקריפטים
- `scripts/query_datagov.py` — חיפוש מערכי נתונים, בדיקת משאבים והרצת שאילתות SQL/datastore ישירות מול ה-API של data.gov.il (CKAN) משורת הפקודה. תומך בפקודות משנה: `search`, `dataset`, `query`, `sql`, `orgs`. הרצה: `python scripts/query_datagov.py --help`

### חומרי עזר
- `references/ckan-api-reference.md` — קטלוג מלא של נקודות קצה עבור ה-API של data.gov.il (CKAN) כולל פרמטרי חיפוש, תחביר שאילתות datastore, יכולות SQL ומזהי ארגונים נפוצים. היעזרו בו בעת בניית קריאות API או איתור באגים בתחביר שאילתות.

## פתרון בעיות

### שגיאה: "Dataset not found"
סיבה: מונחי חיפוש ספציפיים מדי או בשפה לא נכונה
פתרון: נסו מילות מפתח רחבות יותר בעברית. נתונים ממשלתיים הם בעיקר בעברית.

### שגיאה: "Datastore not available"
סיבה: לא לכל המשאבים יש API של datastore (לשאילתות) מופעל
פתרון: הורידו את קובץ ה-CSV/Excel ישירות ועבדו עליו מקומית.

### שגיאה: "Hebrew field names"
סיבה: רוב מערכי הנתונים הממשלתיים משתמשים בשמות עמודות בעברית
פתרון: בצעו שאילתה ראשונה עם limit=1 כדי לראות את כל שמות השדות, ואז בנו שאילתות ממוקדות.
