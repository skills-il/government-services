# Israel Land Authority (RMI) System Guide

## Israeli Land System
- ~93% of land is state-owned (Israel Land Authority)
- ~7% privately owned (registered in Tabu)
- Leasehold model: 49-98 year leases
- 2009 reform: Allows conversion to ownership (residential)

## Tender Types
| Type | Hebrew | Description |
|------|--------|-------------|
| Public tender | michraz | Competitive bidding |
| Lottery | hagrala | Random selection, fixed price |
| Price buyer | mechir larocheish | Fixed price, eligibility-based |
| Direct allocation | haktzaah yeshira | For public institutions |
