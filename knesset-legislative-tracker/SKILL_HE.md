# מעקב חקיקה בכנסת

## הוראות

### שלב 1: זיהוי השאילתה החקיקתית

שאלו את המשתמש מה הוא רוצה לעקוב אחריו:

| סוג שאילתה | נקודת קצה ב-API | מונח בעברית |
|-----------|----------------|-------------|
| חיפוש הצעות חוק | /api/v2/bills | הצעת חוק |
| מצב הצעת חוק | /api/v2/bills/{id} | מצב הצעה |
| ישיבות ועדות | /api/v2/committees | ועדה |
| מידע על חברי כנסת | /api/v2/members | חבר כנסת |
| הצבעות | /api/v2/votes | הצבעה |
| ישיבות מליאה | /api/v2/plenum | מושב מליאה |

הבהירו:
- **באיזה נושא?** (הצעת חוק ספציפית, תחום מדיניות, פעילות חבר כנסת)
- **באיזה כנסת?** (כנסת נוכחית או כנסת היסטורית ספציפית מהכנסת ה-18 ואילך)
- **מה הפלט הרצוי?** (עדכון סטטוס, פירוט הצבעות, ציר זמן, ניתוח מגמות)

### שלב 2: שאילתה ב-API הפתוח של הכנסת

כתובת בסיס: `https://main.knesset.gov.il/Activity/Info/`

הכנסת מספקת API מסוג REST תואם OData:

**חיפוש הצעות חוק לפי מילת מפתח:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/bills?$filter=contains(Name,'מילת_מפתח')&$top=20
```

**קבלת פרטי הצעת חוק:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/bills(BILL_ID)
```

**רשימת ישיבות ועדה:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/committees?$filter=TypeDesc eq 'שם_ועדה'
```

**קבלת רשומת הצבעות של חבר כנסת:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/members(MEMBER_ID)/votes
```

**סינון לפי טווח תאריכים:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/bills?$filter=LastUpdatedDate ge 2024-01-01T00:00:00Z
```

**טיפים:**
- השתמשו ב-`$filter`, `$select`, `$top`, `$skip`, `$orderby` של OData לשאילתות
- שמות הצעות חוק הם בעברית -- חפשו עם מילות מפתח בעברית
- ה-API מחזיר JSON כברירת מחדל
- דפדוף: השתמשו ב-`$top` ו-`$skip` עבור קבוצות תוצאות גדולות

### שלב 3: מעקב אחר התקדמות הצעת חוק דרך הקריאות

החקיקה הישראלית עוקבת אחר תהליך מוגדר:

| שלב | עברית | סטטוס API | תיאור |
|-----|-------|----------|-------|
| הצעה | הצעה | Proposed | טקסט ראשוני של הצעת חוק הוגש |
| הצבעה מוקדמת | הצבעה מוקדמת | PreVote | הצבעה מוקדמת בכנסת |
| ועדה | ועדה | InCommittee | דיון ותיקונים בוועדה |
| קריאה ראשונה | קריאה ראשונה | FirstReading | הצבעת מליאה על עקרונות |
| הכנה לקריאה שנייה | הכנה לקריאה שנייה | PrepSecondReading | תיקוני ועדה סופיים |
| קריאה שנייה ושלישית | קריאה שנייה ושלישית | SecondThirdReading | הצבעה סופית, הופך לחוק |
| אושר | אושר | Approved | נחקק כחוק |
| נדחה | נדחה | Rejected | ההצעה נדחתה |

**מעקב אחר הצעת חוק ספציפית דרך השלבים:**
```python
import requests

bill_id = 12345
url = f"https://main.knesset.gov.il/Activity/Info/api/v2/bills({bill_id})"
response = requests.get(url)
bill = response.json()

print(f"הצעת חוק: {bill.get('Name')}")
print(f"סטטוס: {bill.get('StatusTypeDesc')}")
print(f"עדכון אחרון: {bill.get('LastUpdatedDate')}")
print(f"מציעים: {bill.get('InitiatorFirstName')} {bill.get('InitiatorLastName')}")
```

### שלב 5: ניתוח רשומות הצבעה של חברי כנסת

בדיקת פעילות ודפוסי הצבעה של חברי כנסת:

**קבלת רשימת חברי כנסת נוכחיים:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/members?$filter=IsCurrent eq true
```

**קבלת פרטי חבר כנסת ספציפי:**
```
GET https://main.knesset.gov.il/Activity/Info/api/v2/members(MEMBER_ID)
```

**ניתוח דפוסי הצבעה:**
1. אחזרו רשימת הצבעות להצעת חוק או ישיבה
2. הצליבו עם שיוך מפלגתי
3. זהו דפוסי הצבעה של קואליציה מול אופוזיציה
4. עקבו אחר שיעורי היעדרות והימנעות

**שיוכי מפלגות (סיעות):**
| מפלגה | עברית | קואליציה/אופוזיציה |
|-------|-------|-------------------|
| ליכוד | ליכוד | משתנה לפי כהונה |
| יש עתיד | יש עתיד | משתנה לפי כהונה |
| ש"ס | ש"ס | משתנה לפי כהונה |
| יהדות התורה | יהדות התורה | משתנה לפי כהונה |
| האחדות הלאומית | האחדות הלאומית | משתנה לפי כהונה |

### שלב 6: התראות חקיקה למגזר הטכנולוגיה

מעקב אחר חקיקה רלוונטית לתעשיית הטכנולוגיה:

**מילות מפתח לחיפוש חקיקת טכנולוגיה:**
| נושא | מילות מפתח בעברית | מילות מפתח באנגלית |
|------|-------------------|-------------------|
| פרטיות/הגנת מידע | הגנת פרטיות, מידע אישי | privacy, personal data |
| אבטחת סייבר | סייבר, אבטחת מידע | cyber, information security |
| רגולציית AI | בינה מלאכותית, AI | artificial intelligence |
| עבודה מרחוק | עבודה מרחוק, דיני עבודה | remote work, labor law |
| קניין רוחני | קניין רוחני, פטנטים | IP, patents |
| הטבות מס | הטבות מס, אנגל | tax benefits, angel law |
| פינטק | פינטק, שירותים פיננסיים | fintech, financial services |

**דוגמה: חיפוש הצעות חוק טכנולוגיות אחרונות:**
```python
import requests

keywords = ["סייבר", "פרטיות", "בינה מלאכותית", "טכנולוגיה"]
for kw in keywords:
    url = f"https://main.knesset.gov.il/Activity/Info/api/v2/bills"
    params = {"$filter": f"contains(Name,'{kw}')", "$top": 5, "$orderby": "LastUpdatedDate desc"}
    resp = requests.get(url, params=params)
    bills = resp.json().get("value", [])
    for bill in bills:
        print(f"[{bill['StatusTypeDesc']}] {bill['Name']}")
```

### שלב 7: הצגת ממצאים

עצבו את התוצאות עבור המשתמש:
1. **סיכום הצעת חוק**: שם, מציע/ים, סטטוס נוכחי, פעולה הבאה הצפויה
2. **ציר זמן**: תאריכים מרכזיים מההצעה ועד השלב הנוכחי
3. **פירוט הצבעות**: ספירת בעד/נגד/נמנע עם פילוח מפלגתי
4. **הערות ועדה**: נקודות דיון ותיקונים רלוונטיים
5. **הערכת השפעה**: ניתוח קצר של משמעות הצעת החוק לתחום המשתמש

## דוגמאות

### דוגמה 1: מעקב אחר הצעת חוק ספציפית
המשתמש אומר: "מה הסטטוס של הצעת חוק הגנת הפרטיות?"
פעולות:
1. חיפוש הצעות חוק: `$filter=contains(Name,'הגנת פרטיות')`
2. קבלת פרטי הצעת החוק האחרונה התואמת
3. בדיקת שלב נוכחי (ועדה, קריאה וכו')
4. רשימת חברי כנסת מציעים ושיוך מפלגתי
תוצאה: סטטוס הצעת חוק עם ציר זמן ופעולה הבאה הצפויה

### דוגמה 2: ניתוח רשומת הצבעות של חבר כנסת
המשתמש אומר: "איך הצביע ח"כ X על הצעות חוק בתחום הטכנולוגיה בכנסת הנוכחית?"
פעולות:
1. מציאת חבר הכנסת לפי שם ב-API של חברי כנסת
2. אחזור רשומת הצבעות לכהונת הכנסת הנוכחית
3. סינון הצבעות על הצעות חוק בתחום הטכנולוגיה (סייבר, פרטיות, AI)
4. חישוב סטטיסטיקות דפוסי הצבעה
תוצאה: טבלת סיכום הצבעות עם פירוט בעד/נגד

### דוגמה 3: מעקב אחר ישיבות ועדה
המשתמש אומר: "מה דנה ועדת המדע והטכנולוגיה בחודש האחרון?"
פעולות:
1. שאילתת API של ועדות מסוננת לפי ועדת המדע והטכנולוגיה
2. סינון ישיבות לפי טווח תאריכים (30 יום אחרונים)
3. אחזור סדרי יום ונושאי הצעות חוק שנדונו
4. סיכום נושאים מרכזיים והחלטות
תוצאה: רשימת ישיבות עם סעיפי סדר יום ותוצאות

### דוגמה 4: התראת חקיקת טכנולוגיה
המשתמש אומר: "האם יש הצעות חוק חדשות בנושא רגולציית AI?"
פעולות:
1. חיפוש הצעות חוק עם מילות מפתח: "בינה מלאכותית", "AI", "אלגוריתם"
2. סינון לפי הצעות אחרונות (90 יום אחרונים)
3. בדיקת סטטוס כל הצעת חוק תואמת
4. זיהוי שיוכי ועדות והצבעות קרובות
תוצאה: רשימת הצעות חוק פעילות בתחום AI עם סטטוס ומציעים

## פתרון בעיות

### שגיאה: "לא הוחזרו תוצאות"
סיבה: מונחי חיפוש עשויים שלא להתאים בדיוק לשמות הצעות חוק בעברית
פתרון: נסו מילות מפתח רחבות יותר בעברית. שמות הצעות חוק משתמשים בעברית משפטית פורמלית שעשויה להיות שונה מהשפה היומיומית.

### שגיאה: "חריגה ממגבלת קצב API"
סיבה: יותר מדי בקשות בפרק זמן קצר
פתרון: הוסיפו השהייה של שנייה אחת בין קריאות API. השתמשו ב-`$top` ו-`$skip` לדפדוף במקום אחזור כל הרשומות.

### שגיאה: "חבר כנסת לא נמצא"
סיבה: איות שם חבר הכנסת עשוי להשתנות בין עברית ותעתיק
פתרון: חפשו לפי שם חלקי או השתמשו בנקודת הקצה של רשימת חברי הכנסת כדי למצוא תחילה את המזהה הנכון.

### שגיאה: "נתוני הצבעה לא זמינים"
סיבה: לא לכל ההצבעות יש פירוט לפי חבר כנסת ב-API
פתרון: בדקו את אתר הכנסת ישירות עבור רשומת הצבעת מליאה הספציפית.
